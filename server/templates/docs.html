Quickstart Guide

Deploy your first global endpoint in under 60 seconds. GoInstant provides a seamless developer experience to manage domains, webhooks, and serverless files with a single command.

download

Getting Started

Install the GoInstant CLI globally using npm or your preferred package manager to begin your journey.

zsh — npm
$ npm install -g @goinstant/cli
terminal

CLI Command Reference

Manage your deployments and local tunnels directly from your terminal.

goinstant expose <port>

Creates a secure public URL for your local development server.

Argument Description Required
port The local port to tunnel Yes
--subdomain Request a custom named subdomain No

goinstant deploy

Uploads your current directory to the edge network for global distribution.

Flag Description Default
--prod Promote deployment directly to production false
--env Override environment variables null
sdk

SDK Integration

import { GoInstant } from '@goinstant/sdk';

// Initialize the client
const client = new GoInstant({
  apiKey: process.env.GOINSTANT_KEY,
  region: 'us-east-1'
});

// Expose a local hook
await client.hooks.create({
  target: 'http://localhost:3000',
  events: ['payment.success']
});
